# MONEY-CEO

GitHub Pages **preview** for Money CEO.  
> The full PHP app (login, budgets, debts, DB) runs on cPanel or any PHP host.  
> This repo’s `/docs` folder shows a static UI demo for quick preview.

## How to publish the preview
1. Go to **Settings → Pages**.
2. Source: **Deploy from a branch** → Branch **main** → Folder **/docs**.
3. Save and open `https://<your-username>.github.io/MONEY-CEO/`.

## Notes
- Pages only serves static files (HTML/CSS/JS). No PHP/MySQL here.
- The real app lives outside `/docs` and should be deployed to your hosting.
- Colors: charcoal/dark, white text, red accent; clean dashboard UI.
